Central de Ajuda

Bem-vinda à Central de Ajuda do Unilink! Aqui você encontra informações sobre como usar o aplicativo.

1. Como fechar uma carta de usuário aberta?

Para fechar uma carta após abri-la (virá-la), o usuário pode:

  • Clicar novamente na carta para que ela volte ao estado anterior;
  • Clicar na carta novamente embaixo;
  • Voltar para o menu ou para a tela anterior utilizando o botão de Usuários no Menu.

2. Como adicionar um amigo?

  1. Acesse a aba “Usuários”;
  2. Clique em uma das cartas disponíveis;
  3. Utilize os botões localizados na frente ou no verso da carta para enviar a solicitação de amizade.

3. Como atualizar ou embaralhar as cartas?

  1. Acesse a tela de usuários;
  2. Clique no botão de “Atualizar”.

Isso recarregará a lista de cartas e trará novos usuários aleatoriamente.

4. Como usar o Unibot?

O Unibot é o assistente virtual da plataforma. Para utilizá-lo:

  1. Acesse a aba de mensagens (DM);
  2. Clique no ícone correspondente ao chatbot;
  3. Digite os comandos disponíveis:
    • “1” para visualizar o cardápio;
    • “2” para visualizar o horário das aulas;
  4. Clique em “Enviar”.

5. Como postar um story?

  1. Acesse a Home;
  2. Clique no ícone de criação de stories;
  3. Escolha entre tirar uma foto/vídeo ou selecionar conteúdo da galeria;
  4. Adicione textos, figurinhas ou mencione amigos;
  5. Clique em “Publicar story”.

6. Como interagir com um story?

Com um story aberto:

  1. Clique na lateral direita ou esquerda para pular;
  2. Para responder, clique no campo de mensagem, digite e envie;
  3. Para reagir com emoji, clique no botão de adicionar emoji, selecione e envie.

7. Como descobrir meu UniMatch?

  1. Acesse a aba “Quiz”;
  2. Clique em “Iniciar o Quiz”;
  3. Responda todas as perguntas;
  4. Raspe a carta;
  5. Clique em “Resultado”.

8. Como confirmar presença em um evento?

  1. Vá até a aba “Eventos”;
  2. Clique em “Mais detalhes” no evento desejado;
  3. Clique em “Confirmar presença”.

9. Como visualizar o mapa de um evento?

  1. Acesse a aba “Eventos”;
  2. Clique em “Mais detalhes” do evento;
  3. Clique na opção de visualizar a localização no mapa.

10. Como enviar uma mensagem?

  1. Clique no ícone de mensagens diretas (DM);
  2. Selecione o contato ou abra um novo chat;
  3. Digite a mensagem na barra correspondente;
  4. Clique em “Enviar”.

11. Como solicitar uma chamada?

  1. Acesse o chat com a pessoa desejada;
  2. Clique no ícone de ligação localizado no topo da tela.

12. Como editar minha biografia?

  1. Acesse o seu perfil;
  2. Clique no botão “Mostrar/Editar biografia”;
  3. Clique no campo de texto desejado para alterar;
  4. Após as alterações, clique em “Concluir”.

13. Como vender no Joga pra Rolo?

  1. Acesse a aba “Joga pra Rolo”;
  2. Clique no botão “+” para adicionar um novo produto;
  3. Preencha os campos com preço, descrição e imagem;
  4. Clique em “Publicar”.

14. Como comprar no Joga pra Rolo?

  1. Acesse a aba “Joga pra Rolo”;
  2. Clique no produto desejado;
  3. Combine os detalhes da compra com o vendedor por mensagem;
  4. Pague via QR Code (se aplicável);
  5. Aguarde o recebimento do produto.
